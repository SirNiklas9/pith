pith — how context works

Every box on this page except the purple one is deterministic — tree-sitter parses, name resolution, byte math. No AI runs anywhere in the pipeline until the final assembled prompt crosses the boundary.

2 · Scopes — how far resolution may reach

A scope is a permission boundary, not a search hint. uses resolves names only inside the ring you named — it never silently widens.

  • selection — the lines you highlighted. Always sent; never needs permission.
  • uses — resolve only against declarations in this file.
  • uses:dir — …and every file sitting next to it (your settings/ case).
  • uses:project — …anywhere under the enclosing git root. The hard outer wall.
  • opt-in, every time — default is none; a ring is only crossed because you picked it for this one edit.
